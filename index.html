Projects

Mandatory Energy Efficiency Labelling Scheme (MEELS)

Details

The aim of this project was to facilitate the public in choosing energy efficient appliances and raise public awareness on energy saving in the project MEELS, I was responsible for the web design of MEELS using Dreamweaver, HTML and CSS, user interface experience and maintenance. The website was used for public enquiry of the energy efficiency performance and responsive on multi-devices such as tablet, mobile and laptop.

Building Energy Efficiency Scheme (BEEO)

Details

BEEO is a government project to promote building energy efficiency according to the Buildings Energy Efficiency Ordinance. I was responsible for the graphic design of the BEEO materials. This included leaflets, posters and pamphlets, web page design and implemented on multi devices.

Safe Park Safe Use of Amusement Rides (SPSUAR)

Details

SPSUAR is a public education website to promote the safety practices and the safe use of amusement rides. I implemented the computer drawing of the website using Adobe Illustrator, HTML and CSS.

Amusement Ride Database System (ARDS)

ARDS is a system for authorising amusement ride, recording survey report and certificating professional personals. I revamped the project from visual basic format to webpage. I coordinated between users and contractors, and managed checkpoint meetings and tasks.
